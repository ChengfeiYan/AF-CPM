# AF-CPM

AF-Constraint Propagation Mapping 

![](https://github.com/ChengfeiYan/AF-CPM/blob/main/AF-CPM.png)

The pair representations generated by each layer of the 48-layer Evoformer stack are extracted and transform into a distance map using  Distogram Head. The distance map is then normalized to represent the probability of the $C_\beta$ atom distance between two predicted residues at each bin. Afterward, the probabilities for the first 32 bins of the distance distribution map are summed, and the distance distribution map is transformed into a contact probability maps  with a 12Г… threshold. These contact probability maps  effectively reflect the modeling process of the complex within the model.

## Usage

1. Replace the following files in the `openfold/openfold/data` directory with the ones provided in the `data` folder:
   - `data_pipeline.py`
   - `feature_processing_multimer.py`

2. Replace the following files in the `openfold/openfold/model` directory with the ones provided in the `model` folder:
   - `evoformer.py`
   - `model.py`

3. Replace the `run_pretrained_openfold.py` file in the `openfold` root directory with the provided version.

4. Place `run.py` in the same directory as `run_pretrained_openfold.py`, then execute `run.py` to save and transform pair representations.
